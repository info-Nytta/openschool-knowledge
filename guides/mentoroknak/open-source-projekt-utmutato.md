# Open source projekt indítása

Ha open source projektet szeretnél indítani vagy kontribútálni egybe, ezek a legjobb kiindulópontok:

- **[opensource.guide](https://opensource.guide/)** — átfogó útmutató (README, licenc, közösség, karbantartás)
- **[choosealicense.com](https://choosealicense.com/)** — licencválasztó (ha nem tudod melyiket, válaszd az MIT-t)
- **[conventionalcommits.org](https://www.conventionalcommits.org/)** — commit üzenet formátum
- **[semver.org](https://semver.org/)** — verziókezelés (MAJOR.MINOR.PATCH)
- **[github.com/github/gitignore](https://github.com/github/gitignore)** — `.gitignore` sablonok nyelvek szerint

Az OpenSchool saját konvencióit a [CONTRIBUTING.md](../../CONTRIBUTING.md) tartalmazza.

## Hogyan működik a tanulás?

1. **Olvasd el a heti leckét** — minden héthez tartozik egy lecke (elméleti háttér + kódpéldák)
2. **Fogadd el a feladatot** — a GitHub Classroom link létrehozza a saját repódat
3. **Dolgozz a feladaton** — VS Code-ban írd meg a kódot, commitold, és pushold
4. **Ellenőrizd az eredményt** — a GitHub Actions automatikusan futtatja a teszteket, a zöld pipa ✅ = kész
5. **Kérdezz, ha elakadsz** — a Discord `#segítség` csatornán bármikor tudsz kérdezni

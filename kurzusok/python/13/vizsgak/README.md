| Feladat | Pont | Leírás |
|---------|------|--------|
| 1. feladat | 10 | Pydantic sémák (Create/Read/Update) + utility függvények |
| 2. feladat | 20 | SQLAlchemy DB + CRUD API végpontok (query paraméterek, hibakezelés, adatfájl betöltés) |
| 3. feladat | 10 | JWT autentikáció (regisztráció, login, védett végpontok) |
| 4. feladat | 10 | Adatfeldolgozás (feldolgozas.py – 5 függvény) |
| 5. feladat | 10 | Docker (Dockerfile, requirements.txt) + pytest tesztek (8 teszt) |

## Összefoglaló

- **Időtartam:** 13 hét (0–12), heti 2 óra
- **Nyelv:** Python 3.10+
- **Vizsga:** 40 pont, 90 perc, 3 feladat (8 + 14 + 18 pont)
- **Beadás:** GitHub Classroom (automatikus értékelés)
- **Variánsok:** 4 db (A – filmek, B – könyvek, C – zenék, D – sportolók)

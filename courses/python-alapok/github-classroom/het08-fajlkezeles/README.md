# 8. hét – Fájlkezelés

> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

Ezen a héten megtanulod, hogyan olvashatsz és írhatsz fájlokat Pythonban. A fájlkezelés nélkülözhetetlen, ha adatokat szeretnél elmenteni vagy betölteni – például szöveges fájlokból, CSV-ből vagy naplófájlokból.

---

## 8.1 – Fájl létrehozás ⭐
Írj programot (`feladat1.py`), amely létrehoz egy `naplo.txt` fájlt, és beleír 3 sort (pl. a mai nap eseményei).

## 8.2 – Fájl olvasás ⭐
Írj programot (`feladat2.py`), amely beolvassa a `naplo.txt` fájlt és kiírja a tartalmát sorszámozva.

## 8.3 – Szó számláló ⭐⭐
Írj programot (`feladat3.py`), amely beolvassa a `minta.txt` fájlt, és kiírja:
- A sorok számát
- A szavak számát
- A karakterek számát (szóközök nélkül)

## 8.4 – Tanulói adatok ⭐⭐
Írj programot (`feladat4.py`), amely beolvassa a `tanulok.txt` fájlt és kiírja az adatokat formázottan.

## 8.5 – Jegyzetfüzet alkalmazás ⭐⭐⭐
Írj programot (`feladat5.py`), amely:
1. Bekér egy sort a felhasználótól
2. Hozzáfűzi (`"a"` mód) a `jegyzetek.txt` fájlhoz dátummal (pl. „2026-03-09: szöveg")
3. Ki tudja listázni az összes korábbi jegyzetet
4. A program menüből vezérelve fusson (1 = új jegyzet, 2 = listázás, 3 = kilépés)

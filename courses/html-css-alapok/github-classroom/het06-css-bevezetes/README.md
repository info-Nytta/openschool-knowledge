# 6. hét – CSS bevezetés

> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

Ezen a héten áttérünk a HTML szerkezetről a vizuális megjelenésre: megismerkedsz a CSS-sel. Megtanulod az inline, belső és külső stíluslapok közti különbséget, és a szelektorok segítségével célzottan formázod az elemeket.

---

## 6.1 – Inline stílusok ⭐
Készíts egy weboldalt (`feladat1.html`) legalább 5 elemmel, mindegyiken különböző inline stílussal (`style=""`):
- Egy `<h1>` szín beállítással (`color`)
- Egy `<p>` háttérszínnel (`background-color`)
- Egy `<p>` betűmérettel (`font-size`)
- Egy elem igazítással (`text-align: center`)
- Egy elem félkövér stílussal (`font-weight: bold`)

## 6.2 – Belső stíluslap ⭐
Készíts egy weboldalt (`feladat2.html`) belső stíluslap használatával (`<style>` a `<head>`-ben). Formázz legalább 3 különböző HTML elemet elem szelektorral (pl. `h1`, `p`, `body`).

## 6.3 – Külső CSS fájl ⭐⭐
Készíts egy weboldalt (`feladat3.html`) és egy külső stílusfájlt (`feladat3.css`). Kösd be a CSS fájlt `<link>` elemmel! A CSS fájl tartalmazza:
- `body` háttérszín és betűtípus beállítás
- `h1` szín és igazítás
- `p` sortávolság (`line-height`) beállítás

## 6.4 – Osztály szelektorok ⭐⭐
Készíts egy weboldalt (`feladat4.html`) és stílusfájlt (`feladat4.css`). Hozz létre legalább 3 CSS osztályt (`.kiemelt`, `.figyelmeztetes`, `.info`) és használd őket különböző elemeken. Minden osztálynak legyen:
- Háttérszín
- Szövegszín
- Szegély (`border`)

## 6.5 – Színek és betűtípusok ⭐⭐
Készíts egy weboldalt (`feladat5.html`) és stílusfájlt (`feladat5.css`), amelyben bemutatod:
- Legalább 3 különböző színmegadási módot: névvel, hex-szel, RGB-vel
- Legalább 2 különböző betűtípust (`font-family`)
- `text-decoration` és `text-transform` használatát

## 6.6 – Formázott cikk oldal ⭐⭐⭐
Készíts egy weboldalt (`feladat6.html`) és stílusfájlt (`feladat6.css`):
- Az oldal legyen egy formázott cikk (címsor, bekezdések, idézet, lista)
- Használj osztály és azonosító (`#`) szelektorokat
- Legyen formázott idézet (`blockquote`) saját stílussal
- Betűtípus, szín, háttér, igazítás, sortávolság mind legyen beállítva
- Az egész oldal legyen vizuálisan igényes

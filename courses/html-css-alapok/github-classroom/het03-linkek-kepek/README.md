# 3. hét – Linkek és képek

> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

Ezen a héten megtanulod, hogyan helyezhetsz el linkeket és képeket a weboldalaidon. A hiperhivatkozások (`<a>`) és a képek (`<img>`) a web alapvető építőkövei – ezek nélkül nem létezne az internet, ahogy ismerjük.

---

## 3.1 – Kedvenc oldalak ⭐
Készíts egy weboldalt (`feladat1.html`), amely legalább 5 kedvenc weboldaladra mutató linket tartalmaz (`<a href="">`). Minden link mellett legyen egy rövid leírás.

## 3.2 – Képgaléria ⭐
Készíts egy weboldalt (`feladat2.html`) legalább 3 képpel (`<img>`). Minden képnek legyen:
- `src` attribútum (használhatsz placeholder képeket, pl. `https://picsum.photos/300/200`)
- `alt` szöveges leírás
- `width` méret beállítás

## 3.3 – Navigációs menü ⭐⭐
Készíts egy weboldalt (`feladat3.html`) navigációs menüvel:
- Legalább 4 menüpont, amelyek az oldal belsejére mutatnak (`#szekci1`, `#szekci2`, stb.)
- Minden szekciónak legyen `id` attribútuma
- Minden szekció legalább 1 bekezdés szöveget tartalmazzon

## 3.4 – Képes cikk ⭐⭐
Készíts egy weboldalt (`feladat4.html`), amely egy rövid cikket tartalmaz legalább 2 képpel. Használj:
- `<figure>` és `<figcaption>` a képekhez
- `<a>` linkeket külső forrásokra (`target="_blank"`)
- Címsorokat és bekezdéseket a szöveghez

## 3.5 – Többoldalas mini-weboldal ⭐⭐⭐
Készíts egy 3 oldalas mini-weboldalt:
- `index.html` – Főoldal, linkek a másik két oldalra
- `rolam.html` – Rólam oldal, kép magadról (vagy placeholder kép), és link vissza a főoldalra
- `kapcsolat.html` – Kapcsolat oldal, email link (`mailto:`), és link vissza a főoldalra

Minden oldalon legyen navigációs menü a többi oldalra mutató relatív linkekkel!

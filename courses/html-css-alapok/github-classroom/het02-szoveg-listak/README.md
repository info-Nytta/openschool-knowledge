# 2. hét – Szöveg és listák

> **Beadás:** commitold és pushold a megoldásaidat ebbe a repoba!
> A feladatok nehézség szerint vannak jelölve: ⭐ könnyű | ⭐⭐ közepes | ⭐⭐⭐ nehéz

---

Ezen a héten megtanulod a szövegformázás és a listák használatát HTML-ben. Rendezett és rendezetlen listákat készítesz, és megtanulod, hogyan lehet a tartalmakat áttekinthetően strukturálni.

---

## 2.1 – Kedvenc filmek listája ⭐
Készíts egy weboldalt (`feladat1.html`), amely az 5 kedvenc filmedet tartalmazza rendezetlen listában (`<ul>`, `<li>`).

## 2.2 – Napi teendők ⭐
Készíts egy weboldalt (`feladat2.html`) a napi teendőiddel rendezett listában (`<ol>`, `<li>`). Legalább 6 teendő legyen.

## 2.3 – Tananyag struktúra ⭐⭐
Készíts egy weboldalt (`feladat3.html`) egy tantárgy anyagáról beágyazott listákkal. Például:
```
Matematika
  ├── Algebra
  │   ├── Egyenletek
  │   └── Függvények
  └── Geometria
      ├── Síkidomok
      └── Testek
```
Használj `<ul>` elemeket beágyazva egymásba!

## 2.4 – Idézet oldal ⭐⭐
Készíts egy weboldalt (`feladat4.html`) legalább 3 híres idézettel. Használj:
- `<blockquote>` az idézetekhez
- `<cite>` a szerzőkhöz
- `<p>` az idézetek kontextusához

## 2.5 – Kód bemutató ⭐⭐
Készíts egy weboldalt (`feladat5.html`), amely bemutat legalább 2 programozási kódrészletet. Használj:
- `<code>` inline kódhoz
- `<pre>` kódblokkokhoz
- `<p>` a magyarázatokhoz

## 2.6 – Teljes tartalom oldal ⭐⭐⭐
Készíts egy weboldalt (`feladat6.html`), amely egy komplett „Kedvenc dolgaim" oldal:
- `<h1>` főcímsor
- Legalább 3 szekció, mindegyik `<h2>` címsorral (pl. Filmek, Zenék, Könyvek)
- Minden szekcióban legyen lista (rendezett vagy rendezetlen)
- Legalább 1 `<blockquote>` idézet
- Speciális karakterek használata (`&amp;`, `&lt;`, `&gt;`)
- `<span>` használat szövegrészek megjelölésére

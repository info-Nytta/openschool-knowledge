# Feladatok – 24. hét: Vizsga

> A vizsgán nincs külön feladatlap – a vizsga feladatokat a tanár osztja ki a vizsga napján.
> Készülj fel a 12. lecke próbavizsga feladatával és a korábbi projektek átismétlésével.

---

### Felkészülési feladatok

### 24.1 – Gyors API ⭐⭐
Gyakorold: építs fel egy teljes CRUD API-t a nulláról 30 perc alatt. Pydantic modell, in-memory lista, 5 végpont.

### 24.2 – DB integráció ⭐⭐
Gyakorold: adj hozzá SQLAlchemy-t + Alembic-et a fenti API-hoz 30 perc alatt. CRUD modul, sémák, get_db.

### 24.3 – Auth hozzáadás ⭐⭐
Gyakorold: adj hozzá auth-ot (regisztráció, login, JWT, védett végpontok) 30 perc alatt.

### 24.4 – Tesztek ⭐⭐⭐
Gyakorold: írj legalább 8 tesztet mock DB-vel 30 perc alatt. Auth + CRUD + negatív tesztek.

### 24.5 – Teljes vizsga szimuláció ⭐⭐⭐
Oldd meg a próbavizsga feladatot (12. lecke) szigorú időlimittel: 240 perc, API + DB + Auth + tesztek + Docker.

# 21. hét – Projekt tervezés

> **Határidő:** a következő óra előtt
> **Beadás:** commitold és pushold a megoldásaidat ebbe a repóba!

---

## Feladat

Tervezd meg a záró projektedet.

### 21.1 – Témaválasztás
Válassz: **Todo API** | **Blog API** | **Webshop API** | **Recept API**

Írd bele a `README.md`-ba a választott témát és 2-3 mondatos leírást.

### 21.2 – Adatmodell
Tervezd meg az adatmodellt (legalább 3 tábla, legalább 1 ForeignKey). Dokumentáld a `README.md`-ban.

### 21.3 – Végpont terv
Írd össze a végpontokat táblázatban (módszer, útvonal, leírás, auth). Legalább 10 végpont.

### 21.4 – Projekt setup
Hozd létre: könyvtárstruktúra, `requirements.txt`, `.env.example`, `docker-compose.yml`, `Dockerfile`.

### 21.5 – Első futtatás
`GET /` és `GET /health` működjön. `docker compose up --build` sikeresen elinduljon.

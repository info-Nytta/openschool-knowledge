# 22. hét – Projekt fejlesztés I.

> **Beadás:** commitold és pushold a megoldásaidat ebbe a repóba!

---

Ezen a héten elkezded a záró projekted fejlesztését. Az alapvető CRUD műveletekre, az adatbázis-integrációra és az authentikációra fókuszálj.

---

## Feladat

Fejleszd a záró projektedet.

### Ezen a héten:
- [ ] SQLAlchemy modellek `relationship()`-ekkel
- [ ] Alembic migrációk
- [ ] Pydantic sémák (Create, Update, Response)
- [ ] CRUD modul
- [ ] Routerek minden végponthoz
- [ ] Auth: regisztráció, login, JWT, védett végpontok
- [ ] Swagger UI-ban minden működik

### Minimum követelmény:
- Legalább 3 működő modell
- Teljes CRUD legalább 1 modellhez
- Auth + védett végpontok

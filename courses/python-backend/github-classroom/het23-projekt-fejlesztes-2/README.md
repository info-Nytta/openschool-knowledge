## Feladat

Fejezd be a záró projektedet.

### Ezen a héten:
- [ ] `tests/conftest.py` SQLite mock DB-vel
- [ ] Legalább 15 teszt (auth + CRUD + negatív)
- [ ] `.github/workflows/ci.yml` (lint + test)
- [ ] Docker Compose (`api` + `db`)
- [ ] `README.md` (leírás, végpont táblázat, telepítés, tesztelés)

### Végső ellenőrzőlista:
- [ ] Minden végpont működik (Swagger UI)
- [ ] Min. 15 teszt zöld
- [ ] `docker compose up --build` hibátlan
- [ ] GitHub Actions CI zöld
- [ ] README.md kész
- [ ] `.env.example` létezik
- [ ] Nincs jelszó vagy titkos kulcs a kódban
